## 🚢 部署方案

### Docker容器化部署
```bash
# 一键启动所有服务
docker-compose up -d

# 服务访问地址
- 管理后台: http://localhost:3000
- 商城前端: http://localhost:3001
- 后端API: http://localhost:8080
- MySQL: localhost:3306
- Redis: localhost:6379
```
